# Portfolio Design System

Welcome to the component documentation for this modern portfolio template built with **React**, **TypeScript**, **Vite**, and **CSS Modules**.

## 🎯 Purpose

This design system provides a consistent, reusable set of components for building professional portfolio websites with:
- **Type Safety**: Full TypeScript support
- **Scoped Styling**: CSS Modules for zero conflicts
- **Performance**: Optimized with Vite and lazy loading
- **Accessibility**: WCAG compliant components
- **Dark Mode**: Built-in theme switching

## 📚 Component Organization

### Foundation
Core design tokens and icon library
- **Icons**: SVG icon system with mindstudio and treez logos

### Base Components
Fundamental building blocks for your UI
- **Button**: Primary, ghost, and link button variants
- **Text**: Typography with size and color options
- **Heading**: H1-H6 semantic headings
- **Link**: Internal and external navigation
- **Image**: Responsive images with lazy loading
- **List**: Ordered and unordered lists with variants
- **Container**: Max-width content wrapper
- **Section**: Semantic page sections with IDs

- Header

- Navigation

## 🎨 Design Principles

### CSS Modules & camelCase
All components use CSS Modules with camelCase naming:
```tsx
import styles from './Component.module.css';

<div className={styles.container}>
  <h1 className={styles.title}>Title</h1>
</div>
```

### Data Attributes for Variants
Component variants use data attributes instead of string concatenation:
```tsx
<Button data-variant="primary">Click me</Button>
<List data-variant="compact">...</List>
```

### Mobile-First Responsive
All components are designed mobile-first with responsive breakpoints.

## 🚀 Getting Started

### Installation
```bash
npm install
npm run dev
```

### Using Components
```tsx
import { Button, Heading, Text } from '@/components/base';

function MyPage() {
  return (
    <Container>
      <Heading level={1}>Welcome</Heading>
      <Text size="lg">Get started with these components</Text>
      <Button variant="primary">Get Started</Button>
    </Container>
  );
}
```
