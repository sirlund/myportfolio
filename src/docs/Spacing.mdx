# Spacing

The spacing system provides consistent spacing values across all components using a base unit multiplier approach.

## Base Unit

All spacing is based on a **0.25rem (4px)** base unit, defined as:

```css
--spacing: 0.25rem
```

This creates a predictable and harmonious spacing scale throughout the design system.

## Spacing Scale

## Usage

### In CSS Modules

```css
.button {
  padding: calc(var(--spacing) * 3) calc(var(--spacing) * 6);
  /* 12px vertical, 24px horizontal */
}

.card {
  padding: calc(var(--spacing) * 6);
  gap: calc(var(--spacing) * 4);
  margin-bottom: calc(var(--spacing) * 8);
}

.section {
  padding-top: calc(var(--spacing) * 24);
  padding-bottom: calc(var(--spacing) * 24);
  /* 96px top and bottom */
}
```

### In Inline Styles

```tsx
<div style={{
  padding: 'calc(var(--spacing) * 6)',
  gap: 'calc(var(--spacing) * 4)',
  marginBottom: 'calc(var(--spacing) * 8)'
}}>
  Content
</div>
```

## Grid and Flexbox

The spacing system works seamlessly with CSS Grid and Flexbox:

```css
.grid {
  display: grid;
  gap: calc(var(--spacing) * 6);
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
}

.flex {
  display: flex;
  gap: calc(var(--spacing) * 4);
  flex-wrap: wrap;
}
```

## Mobile Considerations

On mobile devices, consider using smaller spacing values:

```css
.section {
  padding: calc(var(--spacing) * 12);
}

@media (min-width: 768px) {
  .section {
    padding: calc(var(--spacing) * 24);
  }
}
```

## Benefits

- **Consistency**: All spacing follows the same predictable scale
- **Flexibility**: Easy to adjust spacing across the entire design system
- **Mathematical**: Values align perfectly on the 4px grid
- **Maintainability**: Changes to the base unit propagate everywhere
- **Accessibility**: Sufficient spacing improves touch targets and readability
